fn smoothstep(self, edge0: Edge0, edge1: Edge1) -> Self::Output

Performs smooth Hermite interpolation between 0 and 1 when edge0 < self < edge1. smoothstep is equivalent to: t = clamp((x - edge0) / (edge1 - edge0), 0.0, 1.0); return t * t * (3.0 - 2.0 * t); Results are undefined if edge0 ≥ edge1.
